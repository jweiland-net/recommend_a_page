What does it do?
================

This extension uses data collected by `piwik <https://piwik.org>`__ to display pages other visitors also visited. The pages are displayed as links using the typo3 pid.

This extension also works with `realurl <https://typo3.org/extensions/repository/view/realurl/>`__ 2+.
